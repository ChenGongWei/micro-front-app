# micro-front-application

## 启动
启动 react 主应用
```
cd main-react
yarn dev
```
或
```
yarn dev:react
```

启动 vue 子应用
```
cd micro-vue
yarn dev
```
或
```
yarn dev:vue
```

一次性启动
```
yarn dev
```

启动 node 服务
```
cd node
yarn dev
```

## 访问
访问主应用
```
localhost:8088
```

单独访问子应用
```
localhost:9000
```

在主应用访问子应用
```
localhost:8088/vue
```
